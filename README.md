`RichEditor for Flutter` is a beautiful Rich Text `WYSIWYG Editor` for `Flutter` support Android and iOS. .

## Features

Supported Functions
---

- [x] Bold
- [x] Italic
- [x] Subscript
- [x] Superscript
- [x] Strikethrough
- [x] Underline
- [x] Justify Left
- [x] Justify Center
- [x] Justify Right
- [x] Blockquote
- [x] Heading 1
- [x] Heading 2
- [x] Heading 3
- [x] Heading 4
- [x] Heading 5
- [x] Heading 6
- [x] Undo
- [x] Redo
- [x] Indent
- [x] Outdent
- [x] Insert Image
- [x] Insert Youtube
- [x] Insert Video
- [x] Insert Audio
- [x] Insert Link
- [x] Unordered List (Bullets)
- [x] Ordered List (Numbers)

Attribute change of editor
---
- [x] Text Color
- [x] Placeholder
- [x] State Callback

**Milestone**
- [ ] Checkbox
- [ ] Text Background Color
- [ ] Text Font Size
- [ ] Font Family

## Getting started

This will add a line like this to your package's pubspec.yaml (and run an implicit flutter pub get):
    
```yaml
  rich_text_html_editor: ^1.0.2
```

## Usage

See examples in `/example` folder.

```dart
import 'package:rich_text_html_editor/rich_editor.dart';
```
